Django BLACKJACK CARDGAME WEBAPP

This webapp contais the classic casino game of Blackjack within it.

INSTRUCTIONS
- Once on the homepage click "Play Blackjack!" Button, you will be brought to the game screen.
- Your actions are limited to "HIT" or "STAND".
- "HIT" will draw the user a new card.
- "STAND" will concede the users turn.
- If the user or dealer exceed a hand of 21, they automaticaly lose, starting with the player.
- If a users hand closer than, or vice versa, to the value of 21, the one closest will win.
- If both users have the same hand value, they will draw.

FEATURES
- Django SESSION based save/load management
- CLIENT Obfuscation (Inspecting JSON responses will not let you see the deck, or dealer hand)
- UTF-8 Card Graphics
- Restarting games
